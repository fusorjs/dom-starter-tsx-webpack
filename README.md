# Starter

Boilerplate project for:

- [Fusor](https://github.com/fusorjs/dom) DOM library
- [Webpack](https://webpack.js.org/)
- TypeScript

> For JavaScript boilerplate go [here](https://github.com/fusorjs/dom-starter-jsx-webpack)

## Install

```sh
npm install
```

## Start

```sh
npm start
```

## Build

```sh
npm run build
```
